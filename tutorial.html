GIT AND GITHUB TUTORIAL

This page will guide you through all the basics on how to use Git and GitHub for your coding assignments and projects.

Overview

Before we jump into the tutorial on how to use Git and GitHub, let's talk about what Git and GitHub are first.

So what is Git? Git is a local version control system that saves and keeps track of all the changes and different versions of your program in your local machine. This makes it very convenient when we work on our code and projects since we can still see older versions of our program and use them when needed. Furthermore, one of the most useful features of Git is the branching function, which is a function that lets the users make independent branches from their "main" project or code. In a branch, users can test out different methods and ideas for their code without changing or affecting the main file. If the users are satisfied with the work in the branch, they can also merge the branch with the main file so that the changes can also be made to the main code (Devmountain n.d.).

Then what about GitHub? GitHub is a cloud-based platform that hosts and share Git repositories online. Hence, other coders with permission can easily view and contribute to your projects through making changes in different branches. As a result, collaboration on group projects or assignments is so much easier and more convenient (Devmountain n.d.).
